# 🧩 Reducido – Estudio de desarrollo

Portafolio personal que presenta el trabajo y servicios de **Reducido**, un estudio de desarrollo web enfocado en soluciones digitales simples, claras y funcionales.

![Vista previa del sitio](/assets/img/pantallazo01.jpeg)

## 🚀 Demo en vivo

[Ver portafolio en línea](https://tudominio.com)  
_(Actualiza con la URL de tu sitio cuando esté publicada)_

## 📋 Características

- Diseño responsivo con Bootstrap 5
- Carrusel interactivo para mostrar proyectos destacados
- Sección de stack tecnológico con colores personalizados por tecnología
- Navegación suave con offset para barra fija
- Menú colapsable que se cierra automáticamente al hacer clic en un enlace (mejor experiencia en móviles)
- Favicon SVG personalizado con la letra "R" en círculo
- Enlaces directos a GitHub, LinkedIn y correo electrónico (`mailto`)

## 🛠️ Tecnologías utilizadas

| Tecnología       | Descripción                            |

## 🧪 Cómo ejecutar localmente

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Jboschlagos/reducido-portafolio.git
   Abre la carpeta del proyecto y haz doble clic en index.html o usa un servidor local (Live Server de VS Code, por ejemplo).
   ```

No se requieren dependencias adicionales.

✨ Personalización
Imágenes: Reemplaza los archivos en assets/img/ con tus propias imágenes.

Proyectos: Edita el contenido del carrusel en index.html (sección #proyectos).

Stack: Ajusta los colores de cada tecnología en assets/css/style.css (clases .tech-\*).

Contacto: Modifica los enlaces de correo, GitHub y LinkedIn en el pie de página.

🌐 Despliegue
Puedes desplegar este sitio estático en:

GitHub Pages: Ve a Settings > Pages, selecciona la rama principal y guarda.

Netlify: Arrastra la carpeta del proyecto o conecta tu repositorio.

Vercel: Importa el repositorio y publica.

📝 Notas adicionales
El desplazamiento suave y el offset para la barra fija se manejan exclusivamente con CSS (scroll-behavior: smooth y scroll-margin-top).

El menú móvil se cierra automáticamente mediante JavaScript, sin conflictos con Bootstrap.
